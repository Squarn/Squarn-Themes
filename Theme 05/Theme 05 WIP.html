About / Philosophies

I strive to keep my themes as light as possible, using images minimally, only when needed. I don't make themes so that people will end up with exactly the same thing as in the Live Preview. I usually try to make the preview aesthetically pleasing, but the color scheme and images are up to you. I try to make metas work with all types of color options, which can be difficult for ensuring contrast between, say, text and backgrounds. I will use these oppositely in some design elements to make sure that there will be contrast no matter your choices.
